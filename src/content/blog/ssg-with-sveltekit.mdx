date: 2026-07-14
---

A personal homepage does not need a server per request. **Static generation** at build time is enough for the document; the browser can still run canvas, fades, and small interactions.

On okasi.me the landing page prerenders to HTML while the starfield and profile image load run in the client. That is the sweet spot: fast first paint, low hosting cost, no cold starts.

If you are building something similar: ship static, add motion sparingly, and keep the blog in MDX so writing stays close to code.
